# burp-mcp-sse-call-tool

Burp MCP SSE bridge plugin for `jshookmcp`.

## Tools

- `burp_mcp_sse_status`
- `burp_mcp_sse_list_tools`
- `burp_mcp_sse_call_tool`

## Environment

- `BURP_MCP_SSE_URL` (default: `http://127.0.0.1:9876/sse`)
- `BURP_MCP_AUTH_TOKEN` (optional)
- `PLUGIN_BURP_MCP_SSE_CALL_TOOL_ENABLED` / `PLUGINS_BURP_MCP_SSE_CALL_TOOL_ENABLED`
